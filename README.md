NHS Healthcare Performance Dashboard

Project Overview
This project demonstrates the development of an interactive NHS Healthcare Performance Dashboard using Microsoft Excel and Power BI.
It simulates the type of reporting produced by an NHS Information Officer or Performance Analyst to monitor service performance 
and support operational decision-making.

Dashboard Preview


Objectives
.Analyse referral activity across healthcare services.
.Monitor appointment attendance and DNA (Did Not Attend) rates.
.Measure average waiting times by service.
.Compare referral demand across boroughs.
.Monitor achievement of waiting time targets.

Key Findings
.Total Referrals: 505
.Attendance Rate: 71.68%
.DNA Rate: 28.32%
.Average Waiting Time: 28.25 days
.Patients Within Target: 56.63%

Service Performance
.Highest referral demand: Diabetes Clinic and Mental Health (88 referrals each).
.Lowest referral demand: Community Care, Occupational Therapy and Physiotherapy (81 referrals each).
.Shortest average waiting time: Mental Health (20.13 days).
.Longest average waiting time: Occupational Therapy (42.65 days).

Borough Analysis
.Highest referrals: Bracknell (114).
.Lowest referrals: Wokingham (82).
